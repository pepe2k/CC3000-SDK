Welcome To CC3000 Web Server


MSP430-EXP F5529 runs a small memory footprint Web Server. It has capability to host static and dynamic web pages.

In Sample application, Using CC3000 WLAN connectivity, Web Server provides capability to remotely control switching LED’s on/off and displays the status of wheel tick potentiometer mounted on the MSP430F5529board board.
